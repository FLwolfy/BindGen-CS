# BGCS.Runtime

`BGCS.Runtime` is the runtime support library used by generated BindGen-CS bindings.

It contains interop primitives and runtime helpers used by generated code (for example library loading, function tables, pointers, and utility types).

## Repository

- GitHub: https://github.com/FLwolfy/BindGen-CS

## Typical Use

- Reference `BGCS.Runtime` from projects that consume BGCS-generated bindings.
